6. 프로젝트 설명 :
기존의 스트리밍 서비스애는 사용자가 음악을 업로드하거나, YouTube와 연동하여 듣는 기능이 없습니다.
'Music Box'의 줄임말인 MBox는 그러한 문제점을 해결하기 위해 기존 스트리밍 음원과 업로드된 음원을 같은 플레이리스트에 넣어서 들을 수 있도록 개발하였습니다.
MBox는 직관적인 UI로 플레이리스트를 생성하고 음악을 담아서 재생할 수 있다는 장점이 있습니다.
스트리밍 서비스 비용, 가수/연예인 대상으로 하는 활동을 통해 수익 창출 또한 가능합니다.
웹 뿐만아니라 클라우드와 연동하여 앱 또한 개발된다면 누구나 어디서든 사용할 수 있는 서비스로 자리잡을 수 있을 것입니다.

7. 주요 완료 기능 :
- 로그인 및 회원가입 페이지
- 뮤직 플레이어 페이지 (재생중인 항목, 현재 플레이리스트 보기, 재생 기능)
- 플레이리스트 페이지 (플레이리스트 추가, 목록 보기, 재생, 수록곡 편집, 삭제)
- 업로드 페이지 (음원 업로드, YouTube 주소 등록)
- 검색 페이지 (검색 결과 플레이리스트 추가)
- 실시간 차트 (차트 결과 플레이리스트 추가)

8. 미완료 기능 :
- 실 음원 재생 (음원파일 부재로 제한됨)

9. 향후 계획 :
- 이용자간 플레이리스트 공유 기능
- 가수/연예인 대상 페이지 개설 및 소셜 네트워킹 기능
- 개인 음악 방송 기능
